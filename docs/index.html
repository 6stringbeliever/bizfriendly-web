BizFriend.ly is split into two repositories. A lightweight javascript and html front end called bizfriendly-web and a Python based backend called bizfriendl-api.

The reason for the separation is to allow organizations to make their own custom front ends. With a robust API available, we've made it as easy as possible for organizations to use BizFriendly in anyway they need. The posibillites are:

  1. Create content on BizFriend.ly
  2. Clone the bizfriendly website
  3. Make a brand new frontend
  4. Clone the bizfriendly backend
