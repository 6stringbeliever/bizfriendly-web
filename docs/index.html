The Architecture

BizFriend.ly is split into two separate code repositories. The website and the API. We split the code up for several reasons:

  1. Allow for many white labeled frontends.
  2. Easily build new mobile frontends.
  3. Easily redeploy the back end
#

Description of Tech

  • Hosted on Github Pages and Heroku
  • Backend written in Python 2.7.4
  • Uses the Flask microframework
  • Website is simply jQuery
  • We use Jekyll templates
  • Pycco helped me write these docs.
