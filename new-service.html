Use the form below to add a new web service to BizFriend.ly

If you're new to creating content, we recommend checking out the Teaching Guide before you begin.

You'll need to create a BizFriend.ly account before you can create a lesson. Sign up then come back here.
Service Information

To create a lesson about a web service that is new to BizFriend.ly, you'll have to provide some information that helps others know why they should learn about it.

Tips to Keep in Mind

Up to three brief points that learners should keep in mind.

Additional Resources

Add up to five links to additional resources related to how to use the web service.

Service Icon

Add an icon for the web service. It’s important that the icon is 30 pixels by 30 pixels or less. See the teaching guide for more info on icons.

Service Video or Image

Add an image or a video about the web service. The image should be 600 pixels by 338 pixels and either a JPEG or PNG. To add a video, you’ll want to first upload it to Vimeo, set the embed settings to only include the video title and set the size to 600 x 338 pixels, and then copy and paste the entire embed link into the form.

Select A Skill

Select a skill that best fits with your web service. If none of the options look right, use the "add new skill" option in the menu to create a new skill.
